# ホテル予約システム（HRS）チーム課題 要件定義

## 1. 課題概要

ホテル予約システム（HRS: Hotel Reservation System）を題材に、UMLを用いたオブジェクト指向分析・設計から一部実装までをチームで行う演習。

### 1.1 目的

- ドメイン分析
- 要求分析
- システム分析
- UMLによるモデリング

### 1.2 開発対象システム

利用者がホテルを予約し、チェックイン・チェックアウトを行うシステム。

**業務フロー**

1. 利用者が部屋を予約
2. 予約番号を受け取る
3. チェックイン時に予約番号を提示
4. フロントがチェックイン処理
5. 部屋番号を通知
6. 宿泊後チェックアウト
7. 宿泊料金を支払う
